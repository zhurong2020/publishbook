# 英文单词学习书籍

## 项目简介

面向中国英语学习者的词汇学习书籍。

## 目录结构

```
vocab-book/
├── OUTLINE.md              # 书籍大纲
├── README.md               # 本文件
├── content/
│   ├── chapters/          # 各章节内容
│   └── appendices/        # 附录
├── data/
│   ├── vocabulary/        # 单词数据 (JSON)
│   └── exercises/         # 练习题数据
└── assets/
    ├── images/            # 插图
    └── audio/             # 音频（可选）
```

## 书籍规划

### 目标读者
- [ ] 初学者（零基础）
- [ ] 中级学习者（有一定基础）
- [ ] 高级学习者（备考/专业需求）

### 单词来源
- [ ] 高频词汇表
- [ ] 分级词汇（CEFR A1-C2）
- [ ] 考试词汇（托福/雅思/GRE等）
- [ ] 主题词汇（商务/学术/日常等）

### 内容特色
- [ ] 词根词缀记忆法
- [ ] 联想记忆法
- [ ] 情境例句
- [ ] 配套练习
- [ ] 音标与发音指导

## 快速开始

1. 阅读 `OUTLINE.md` 了解书籍结构
2. 在 `data/vocabulary/` 中添加单词数据
3. 在 `content/chapters/` 中编写章节内容
4. 使用 `../../shared/scripts/build.py` 生成输出
