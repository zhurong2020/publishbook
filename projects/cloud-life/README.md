# 普通人的云生活

**副标题**：从零开始，用免费工具打造你的数字王国

## 项目简介

本书面向没有技术背景的普通人，手把手教你搭建属于自己的数字基础设施。不堆砌术语，不罗列工具，只讲你真正需要的。

## 目标读者

- 想摆脱各种订阅费的普通人
- 关心数据隐私但不知从何下手的人
- 对技术好奇但被专业术语吓退的人
- 想用 AI 但担心隐私泄露的人

## 核心理念

1. **普通人视角** - 假设读者只有一台 Windows 电脑
2. **结果导向** - 告诉你"能做什么"，而非"用什么工具"
3. **最小可用** - 每个方案只推荐一个最佳选择
4. **成本透明** - 明确标注免费/付费选项
