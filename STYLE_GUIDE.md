# 写作风格指南

## 一、整体风格

### 语言风格
- 使用简洁明了的中文
- 专业术语首次出现时提供解释
- 避免过于口语化的表达
- 保持前后一致的表述方式

### 格式规范
- 中英文之间添加空格
- 标点符号使用中文全角
- 数字使用阿拉伯数字
- 英文单词使用正确的大小写

## 二、词条编写规范

### 单词选取
1. 优先选取高频实用词汇
2. 同一主题下词汇有逻辑关联
3. 避免过于生僻的词汇（除专门章节）
4. 注意词汇难度的渐进性

### 音标标注
- 使用国际音标 (IPA)
- 同时提供英式和美式发音
- 音标格式：`/fəˈnetɪk/`

### 释义撰写
1. **中文释义**
   - 首选常用义项
   - 按使用频率排列
   - 每个义项不超过 20 字

2. **英文释义**
   - 使用简单词汇解释
   - 来源参考：Longman, Oxford Learner's

### 例句标准
1. **数量要求**：每词至少 2 个例句
2. **难度控制**：使用已学词汇
3. **语境真实**：贴近实际使用场景
4. **长度适中**：10-20 词为宜

**好的例句：**
- She set a good example for her students.
- Can you give me an example of what you mean?

**避免的例句：**
- ❌ Example. （太短，无语境）
- ❌ The quintessential example of... （词汇过难）

### 搭配选取
- 选取最常用的 3-5 个搭配
- 包含不同词性的搭配
- 标注介词搭配

### 记忆技巧
1. **词根词缀法**：ex-(出) + ample(拿)
2. **联想记忆法**：形象联想、谐音联想
3. **词源故事**：简短有趣的来源说明

## 三、练习题规范

### 题目设计原则
1. 测试目标明确
2. 难度适中
3. 选项设计合理（干扰项有意义）
4. 答案唯一确定

### 各题型规范

#### 选择题
- 四选一
- 题干完整清晰
- 干扰项与正确答案词性一致

#### 填空题
- 提供明确语境
- 标注词汇提示或首字母
- 注意词形变化的考查

#### 翻译题
- 中译英为主
- 句子长度适中
- 考查目标词汇突出

## 四、排版规范

### 标题层级
```
# 章标题
## 节标题
### 小节标题
#### 词条标题
```

### 特殊标记
- **粗体**：强调重点
- *斜体*：例句翻译、注释
- `代码格式`：音标、特殊符号
- > 引用：名言、补充说明

### 表格使用
| 场景 | 使用表格 |
|------|----------|
| 词形变化 | ✓ |
| 易混词对比 | ✓ |
| 词汇总结 | ✓ |

### 图表规范
- 图片居中显示
- 添加图片说明
- 使用矢量图优先
